NFT Verification Bot
Discord · Live

Gate your Discord server with real on-chain NFT ownership. Users verify by sending a small pathUSD payment. The bot checks the contract, assigns tiered roles, and monitors wallets hourly to update roles as NFTs move.

Features
On-chain verification / checks ownership directly on Tempo, no middlemen
Custom tiers / any threshold, any role name. You define the structure
Auto role updates / hourly sweep upgrades or removes roles as NFTs move
Multi-server / each server configures their own contract and roles
pathUSD payment / small stablecoin fee proves wallet ownership
Setup
1
Apply for access using the button below
2
Run /setup — set your NFT contract and add tiers
3
Run /panel — post the verification embed in any channel

Sales Bot
Discord · Live

Real-time sale and listing alerts for your NFT collection. Watches Whelmart on Tempo Chain and posts to any Discord channel the moment activity happens.

Features
Instant alerts / sale and listing embeds posted the moment they happen
Collection filter / configure it to watch only your collection
Separate channels / route listings and sales to different channels
Multi-server / each server configures independently
