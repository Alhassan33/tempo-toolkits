NFT Verification Bot
Discord · Tempo mainnet · Live

Gate your Discord server with real on-chain NFT ownership. Users verify by sending a small pathUSD payment from their wallet. The bot checks the NFT contract on Tempo mainnet, assigns tiered roles, and monitors wallets hourly to auto-update roles as NFTs move.

Features
On-chain verification — checks balanceOf directly on Tempo, no middlemen
Custom tiers — any threshold, any role name. You decide the structure
Auto role updates — hourly sweep upgrades or removes roles when NFTs move
Multi-server — each server configures their own contract and roles independently
pathUSD payment — small stablecoin fee proves wallet ownership without a signature
No-code setup — Discord embed wizard, no developers or config files needed
Setup — 3 steps
1
Add the bot to your server using the invite link below
2
Run /setup — set your NFT contract, add tiers with custom thresholds and roles
3
Run /panel — post the verification embed in any channel. Done
